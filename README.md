# 数据中心 BOM 查询与配置器 v1.6

AI 超算数据中心从 0 到 1 建设成本全景分析工具。

## 功能

- **BOM 总览** — 10 大系统投资分布、成本范围可视化
- **BOM 明细** — 78 项设备详情，支持在线编辑价格/数量，实时联动总览
- **价格调研** — 计算/散热/电力/网络四大品类最新市场行情
- **供应商库** — 187 家供应商信息，支持搜索与 CSV 导出
- **更新日志** — 版本迭代记录

## 访问

直接打开 [https://zerlot.github.io](https://zerlot.github.io)

## 本地使用

下载仓库后用浏览器打开 `index.html` 即可，无需任何服务器。

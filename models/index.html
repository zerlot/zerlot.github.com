实质变动 vs 营销包装

声称判定分析
Gemini 3.1 Pro "13/16 基准测试领先" ✅ 实质 ARC-AGI-2 77.1%（上代翻倍），GPQA 94.3% 均为第三方可验证分数。$2/$12 定价确实低于同级竞品。
MiniMax M2.7 "比 Opus 便宜50倍" ⚠️ 部分 对比的是旧版 Opus 4.1（$15/M）。对比当前 Opus 4.6（$5/M），实际便宜 17 倍 input / 21 倍 output。仍然极具性价比。
GPT-5.4 "Computer Use 原生支持" ✅ 实质 OSWorld-Verified 和 WebArena Verified 基准测试均为记录高分。确实是 OpenAI 首个原生 Computer Use 旗舰模型。
“Deep Research/研究代理”只是换名包装 ✅ 实质 Gemini API 已在 4月21日 直接发布可用的新版本 deep-research-preview-04-2026 / deep-research-max-preview-04-2026，并明确新增 MCP server 集成与 File Search（属于可用能力变更，而非纯宣传）。
DeepSeek 宣称“1M 上下文成为默认” ✅ 实质 DeepSeek 官方公告与 API 更新日志同步说明 V4-Pro/V4-Flash 支持 1M context，并作为官方服务默认；同时给出 legacy 模型名退役时间线，属于可操作的产品变更。
小米 MiMo-V2.5 “MIT 开源 + 1M 上下文” ⚠️ 部分 开源与 1M context（以及 MoE 参数规模）已有第三方媒体转述，但在本期仍建议以官方模型卡/公告页进一步核验定价与可直接调用的 API 细节。
Kimi K2.6 “对标 GPT-5.4/Opus 4.6 的 Agentic 基准” ⚠️ 部分 Cloudflare 上线公告披露了 BrowseComp / SWE-Bench Verified / Terminal-Bench 2.0 分数与参数规模，并可直接调用；但更全面基准仍需更多第三方平台统一口径对照。
GLM-5.1 "长程任务可独立持续工作8小时" ⚠️ 部分 智谱官方将其作为旗舰卖点（长任务+tool use），但目前缺少公开第三方基准或可复现实测数据；结论上应按“能力发布已落地，但效果强弱需验证”。
DeepSeek V4 "即将发布" ❌ 拖延 该条为往期记录：目前 DeepSeek 已于 4月24日上架 V4-Pro/V4-Flash，并给出 legacy 模型名退役时间线。后续关注是否发布更明确的“稳定版/后训练版”与更细的定价说明。
Kimi K2.6 "Workers AI Day 0 上线" ✅ 实质 Cloudflare 已提供可直接调用的 @cf/moonshotai/kimi-k2.6，并披露 1T MoE/32B active、262.1k context、multi-turn tool calling，以及 BrowseComp / SWE-Bench Verified / Terminal-Bench 2.0 分数（属于可用能力与接口层面的落地，而非单纯宣传）。
Zhipu GLM-5.1 "最新旗舰" ⚠️ 部分 $1.40/$4.40 定价比 GLM-5 涨 40%，但基准提升幅度不明确。GLM-4.7 仍是性价比最优选。
